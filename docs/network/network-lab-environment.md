# Network Lab Environment

## Recommended Lab Environment

| Component | Recommendation |
|---|---|
| Virtualization | Hyper-V, VMware Workstation, or VirtualBox |
| On-Prem Server | Windows Server VM for AD DS, DNS, DHCP, File Server |
| Client | Windows 10/11 VM domain joined |
| Router / Firewall | pfSense, Windows Server RRAS, or simulated routing |
| Azure | Hub VNet, Spoke VNet, VPN Gateway, Azure Firewall, Private DNS |
| Evidence | Screenshots, command output, logs, and architecture diagrams |

## Minimum Lab

- 1 Windows Server VM
- 1 Windows Client VM
- 1 Azure subscription
- 1 GitHub repo / Docusaurus site for documentation

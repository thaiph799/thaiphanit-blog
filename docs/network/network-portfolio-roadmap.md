# Network Portfolio Roadmap

## Roadmap

| Phase | Focus | Labs |
|---|---|---|
| Phase 1 | On-Prem Foundation | IP addressing, subnetting, VLAN design, DNS, DHCP, AD DS dependency |
| Phase 2 | On-Prem Operations | Domain join, file share access, GPO dependency, VPN, troubleshooting |
| Phase 3 | Hybrid Connectivity | Site-to-site VPN, hybrid DNS, routing, Azure Arc |
| Phase 4 | Azure Network | VNet, NSG, hub-spoke, VPN Gateway, Firewall, Private Link, Load Balancing |
| Phase 5 | Troubleshooting | Network Watcher, route analysis, DNS lookup, connectivity tests |

## Evidence Standard

Each lab should include screenshots, commands, validation results, and a short interview explanation.

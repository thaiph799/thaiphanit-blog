# OSPF Enterprise Routing

## Context

OSPF is used to validate multi-area routing, adjacency stability, summarization boundaries, and operational troubleshooting.

## Validation Commands

```cli
show ip ospf neighbor
show ip ospf database
show ip route ospf
```

## Evidence Checklist

- Neighbor state is stable.
- LSDB is consistent.
- Expected routes are present.
- Failure and recovery behavior is documented.

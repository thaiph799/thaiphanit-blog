# BGP Routing Policy

## Context

BGP labs should show path selection, route advertisement control, filtering, and operational verification.

## Validation Commands

```cli
show bgp summary
show bgp ipv4 unicast
show route-map
```

## Review Notes

Document the business reason for every policy: traffic engineering, route containment, provider boundary, or failover behavior.

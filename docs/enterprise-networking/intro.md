# Enterprise Networking

This track contains network architecture and operations evidence for enterprise routing, switching, security, and observability.

## Focus Areas

| Area | Scope |
|---|---|
| Core Routing & Switching | OSPF, BGP, Layer 2/3 design, Cisco IOS-XE/XR/NX-OS, and Aruba AOS-CX |
| Network Security | Firewall policy, NAT, IPSec VPN, GlobalProtect, and Palo Alto operating patterns |
| Observability | SNMP OID mapping, Syslog, Zabbix, Elasticsearch, SolarWinds, and dashboard evidence |

## Review Path

Start with Core Routing & Switching, then move into Network Security and Observability depending on the role being reviewed.
